# Basics of the Macro Language
 This is a series of imageJ macro files that are written to accompany the short youtube video series about the same topic. Watch the videos first!
 https://www.youtube.com/playlist?list=PLDmdnGCEvwK4
 
 Each file consists of some example code, and a brief set of exercises based on the introduced code. These generally incorporate previously introduced code. The final file (video6_dialogs.ijm) includes an "exam" question, which is answered in "exam_answer.ijm". Don't peek!

 Respectful feedback is appreciated.
